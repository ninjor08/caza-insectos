# Caza Insectos - Taller 1

- Juego desarrollado en React Native con Expo y Firebase.

## Descripción
- Caza Insectos es un videojuego móvil donde el usuario debe atrapar insectos antes de que termine el tiempo.  
- El puntaje final se guarda en Firebase Firestore.

## Tecnologías usadas
- React Native.
- Expo.
- TypeScript.
- Firebase Firestore.

## Funcionalidades
- Juego con temporizador.
- Sistema de puntaje.
- Almacenamiento de puntajes en Firebase.
- Navegación con Expo Router (Tabs).
